USB 网络共享

您可用 USB 电缆将您的 Android 设备网络共享至您的电脑，以将设备的互联网连接与电脑共享
  • USB 网络共享支持 Windows Vista、Windows 7 和 Linux

  • 如果您的设备有 SD 卡或 USB 存储，当您已通过 USB 网络共享时无法将其安装在您的电脑上

  • 欲获取关于包括在其他操作系统上使用 USB 网络共享的更多信息，请访问http://www.android.com/tether#usb

蓝牙网络共享

您可通过蓝牙连接将您的 Android 设备网络共享至您的电脑，以将设备的互联网连接与电脑共享
  • 蓝牙网络共享支持任何电脑或可通过蓝牙访问互联网的其他设备

  • 欲获取更多信息，请访问http://www.android.com/tether#bluetooth
