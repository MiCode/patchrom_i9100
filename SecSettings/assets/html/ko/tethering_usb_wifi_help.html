USB 테더링

USB 케이블로 내 디바이스와 컴퓨터를 연결(테더링)하면 컴퓨터가 디바이스의 인터넷 연결을 공유할 수 있습니다
  • 테더링은 윈도우 7, 윈도우 비스타, 리눅스에서 사용할 수 있습니다

  • 테더링되어 있을 때에는 SD 카드를 컴퓨터에 연결할 수 없습니다

  • 다른 운영체제에서 USB 테더링을 사용하는 방법을 포함하여 더 많은 정보를 얻으려면 http://www.android.com/tether#usb를 방문하세요

휴대용 Wi-Fi 핫스팟

내 디바이스를 휴대용 Wi-Fi 핫스팟으로 전환하여 인터넷 연결을 다른 컴퓨터나 기기와 공유할 수 있습니다
  • 디바이스가 휴대용 Wi-Fi 핫스팟으로 작동 중일 때는 Wi-Fi 연결을 통해 인터넷에 접속하는 애플리케이션을 사용할 수 없습니다

  • Wi-Fi 핫스팟 설정에서 휴대용 Wi-Fi 핫스팟을 설정할 수 있습니다

  • 더 자세한 정보를 보려면 http://www.android.com/tether#wifi를 참조하세요.
