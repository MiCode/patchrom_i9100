USB 테더링

USB 케이블로 안드로이드 디바이스와 컴퓨터를 테더링하면 디바이스의 인터넷 연결을 컴퓨터와 공유할 수 있습니다
  • USB 테더링은 Windows Vista, Windows 7, Linux에서 사용할 수 있습니다

  • 내 디바이스에 SD 카드 또는 내장 메모리가 있으면 USB로 테더링할 때 이를 컴퓨터에 마운트할 수 없습니다

  • 다른 운영체제에서 USB 테더링을 사용하는 방법 등의 자세한 내용은 http://www.android.com/tether#usb를 참조하세요

블루투스 테더링

블루투스 연결로 안드로이드 디바이스와 컴퓨터를 테더링하면 디바이스의 인터넷 연결을 컴퓨터와 공유할 수 있습니다
  • 블루투스 테더링은 블루투스로 인터넷에 접속할 수 있는 모든 컴퓨터 또는 기타 기기에서 사용할 수 있습니다

  • 자세한 내용은 http://www.android.com/tether#bluetooth을 참조하세요
