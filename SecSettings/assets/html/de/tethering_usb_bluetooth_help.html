USB-Tethering

Sie können Ihr Android-Gerät über ein USB-Kabel mit Ihren Computer verbinden und so die Internetverbindung des Geräts für den Computer freigeben.
  • USB-Tethering funktioniert unter Windows Vista, Windows 7 und Linux

  • Wenn Ihr Gerät über eine SD-Karte oder über USB-Speicher verfügt, können Sie diese während des USB-Tethering nicht an den Computer anschließen

  • Weitere Informationen, einschließlich Informationen zum USB-Tethering unter anderen Betriebssystemen, finden Sie unter http://www.android.com/tether#usb

Bluetooth-Tethering

Sie können Ihr Android-Gerät über eine Bluetooth-Verbindung mit Ihren Computer verbinden und so die Internetverbindung des Geräts für den Computer freigeben.
  • Bluetooth-Tethering funktioniert mit Computern und anderen Geräten, die über Bluetooth eine Internetverbindung herstellen können

  • Weitere Informationen unter http://www.android.com/tether#bluetooth
