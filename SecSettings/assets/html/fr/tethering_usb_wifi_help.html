Modem USB

Vous pouvez partager la connexion Internet de votre périphérique avec votre ordinateur en les reliant avec un câble USB.
  • La fonction modem fonctionne avec Windows 7, Windows Vista et Linux

  • Vous ne pouvez pas connecter une carte SD à votre ordinateur lors de la connexion avec un modem

  • Pour obtenir plus d'informations, notamment sur l'utilisation du modem USB avec d'autres systèmes d'exploitation, accédez à l'adresse suivante http://www.android.com/tether#usb

Point d'accès Wi-Fi portable

Vous pouvez convertir votre appareil en point d'accès Wi-Fi pour partager sa connexion Internet avec des ordinateurs et d'autres appareils
  • Impossible d'utiliser des applications pour accéder à Internet via la connexion Wi-Fi pendant que l'appareil sert de point d'accès Wi-Fi portable

  • Vous pouvez configurer le point d'accès Wi-Fi dans les paramètres de point d'accès Wi-Fi

  • Pour plus d'informations, visitez http://www.android.com/tether#wifi
