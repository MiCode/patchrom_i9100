Bærbart Wi-Fi-hotspot

Du kan ændre din telefon til et bærbart Wi-Fi-hotspot for at dele dets internetforbindelse med computere og andre enheder
  • Kan ikke bruge telefonens programmer til adgang til internet via Wi-Fi-tilslutning, mens telefonen fungerer som bærbart Wi-Fi hotspot

  • Du kan konfigurere bærbart Wi-Fi-hotspot i Indstillinger for Wi-Fi-hotspot

  • For flere informationer: Besøg http://www.android.com/tether#wifi

USB-internetdeling

Din computer kan dele din enheds internettilslutning, hvis du binder dem sammen med USB-kabel
  • Binding fungerer med Windows 7, Windows Vista og Linux

  • Du kan ikke slutte SD-kort til din computer, når den er delt

  • For flere informationer, herunder brug af USB-deling med andre operativsystemer: Gå til http://www.android.com/tether#usb

Bluetooth-binding

Du kan binde Android-enheden til din computer via en Bluetooth-forbindelse for at dele enhedens internetforbindelse med computeren
  • Bluetooth-binding fungerer med alle computere eller andre enheder, der kan oprette forbindelse til internettet via Bluetooth

  • Du kan finde flere oplysninger på http://www.android.com/tether#bluetooth
