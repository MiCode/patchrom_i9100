Переносная точка доступа Wi-Fi

Можно использовать устройство в качестве переносной точки доступа Wi-Fi для подключения других компьютеров и устройств к Интернету
  • Если устройство работает в качестве переносной точки доступа Wi-Fi, Интернет через подключение Wi-Fi для приложений устройства недоступен

  • Переносную точку доступа Wi-Fi можно настроить в параметрах точки доступа Wi-Fi

  • Дополнительную информацию см. на веб-сайте http://www.android.com/tether#wifi.

USB-модем

Компьютер может использовать подключение устройства к Интернету, если присоединить устройство к нему в качестве модема с помощью USB-кабеля
  • Функция модема поддерживается в Windows 7, Windows Vista и Linux

  • Подключение карты памяти SD к компьютеру в этом режиме невозможно

  • Дополнительные сведения, в том числе об использовании USB-модема в других операционных системах, см. на веб-сайте http://www.android.com/tether#usb.

Bluetooth-модем

Чтобы предоставить компьютеру доступ к мобильному Интернету, можно подключить к нему устройство под управлением Android с помощью Bluetooth
  • Bluetooth-модем совместим с любыми компьютерами или устройствами, способными использовать мобильный Интернет с помощью Bluetooth

  • Дополнительные сведения см. на веб-сайте http://www.android.com/tether#bluetooth
