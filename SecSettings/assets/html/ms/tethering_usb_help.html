Penambatan USB

Komputer anda boleh berkongsi sambungan Internet foto anda jika anda menambat dengan kabel USB
  • Penambatan dapat berfungsi dengan Windows 7, Windows Vista, dan Linux

  • Anda tidak boleh menyambung kad SD ke komputer anda setelah ditambat

  • Untuk maklumat selanjutnya, termasuk bagaimana mengguna penambatan USB dengan sistem operasi yang lain, pergi ke http://www.android.com/tether#usb
