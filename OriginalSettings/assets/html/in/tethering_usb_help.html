USB penarikan

Anda dapat menambatkan telepon Anda ke komputer dengan kabel USB, untuk berbagi koneksi Internet telepon Anda dengan komputer Anda
  • Penarikan bekerja dengan Windows Vista, Windows 7 dan Linux

  • Anda tidak dapat mount SD card telepon Anda di komputer Anda bila USB-bertambat

  • ntuk informasi lebih lanjut, termasuk menggunakan USB penarikan dengan OS lain, lihat http://www.android.com/tether#usb
