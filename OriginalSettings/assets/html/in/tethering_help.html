USB penarikan

Anda dapat menambatkan telepon Anda ke komputer dengan kabel USB, untuk berbagi koneksi Internet telepon Anda dengan komputer Anda
  • Penarikan bekerja dengan Windows Vista, Windows 7 dan Linux

  • Anda tidak dapat mount SD card telepon Anda di komputer Anda bila USB-bertambat

  • ntuk informasi lebih lanjut, termasuk menggunakan USB penarikan dengan OS lain, lihat http://www.android.com/tether#usb

Wi-Fi penarikan

Anda dapat mengubah ponsel Anda menjadi jalur akses Wi-Fi, untuk berbagi koneksi Internet telepon Anda dengan satu atau lebih komputer atau perangkat lain
  • Bila telepon Anda bertindak sebagai titik akses, Anda tidak dapat menggunakan aplikasi telepon Anda untuk mengakses Internet melalui koneksi Wi-Fi nya

  • Anda mengkonfigurasi jalur akses Wi-Fi Wi-Fi dalam pengaturan penarikan

  • Untuk informasi lebih lanjut, kunjungi http://www.android.com/tether#wifi
