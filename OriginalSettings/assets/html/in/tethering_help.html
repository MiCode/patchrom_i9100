Hotspot Wi-Fi portable

Anda dapat membuat perangkat berfungsi sbg hotspot Wi-Fi portable, untuk membagi koneksi Internetnya dengan komputer atau perangkat lainnya
  • Tidak dapat menggunakan aplikasi perangkat untuk mengakses Internet melalui koneksi Wi-Fi saat perangkat berfungsi sbg hotspot Wi-Fi portable

  • Anda dapat mengonfigurasi hotspot Wi-Fi portable pada pengaturan hotspot Wi-Fi

  • Untuk informasi selengkapnya, kunjungi http://www.android.com/tether#wifi

Penambatan USB

Komputer dapat membagi koneksi Internet perangkat Anda jika Anda menambatkan keduanya dengan kabel USB
  • Penambatan bekerja dengan Windows 7, Windows Vista, dan Linux

  • Anda tidak dapat menyambungkan kartu SD ke komputer Anda saat ditambatkan

  • Untuk informasi lebih lanjut, termasuk bagaimana cara menggunakan penambatan USB dengan sistem pengoperasian lain, masuk ke http://www.android.com/tether#usb

Penambatan Bluetooth

Anda dapat menambatkan perangkat Android Anda ke komputer via koneksi Bluetooth, untuk membagi koneksi Internet perangkat dengan komputer
  • Penambatan Bluetooth berfungsi pada komputer atau perangkat lain yang dapat memperoleh koneksi internet lewat Bluetooth

  • Untuk informasi selengkapnya, lihat http://www.android.com/tether#bluetooth
