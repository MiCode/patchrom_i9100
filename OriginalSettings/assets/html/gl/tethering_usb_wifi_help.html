Ancoraxe da rede USB

O teu ordenador pode compartir a conexión de Internet do teléfono se están ancorados á rede xuntos con un cable USB
  • A ancoraxe á rede funciona con Windows 7, Windows Vista e Linux

  • Non podes conectar unha tarxeta SD ao ordenador cando está ancorado á rede

  • Para máis información, incluíndo como usar a ancoraxe á rede do USB con outros sistemas operativos, visita http://www.android.com/tether#usb

Portable Wi-Fi hotspot

You can turn your device into portable Wi-Fi hotspot, to share its Internet connection with computers or other devices
  • Unable to use device applications to access Internet via Wi-Fi connection while device serves as portable Wi-Fi hotspot

  • You can configure portable Wi-Fi hotspot in Wi-Fi hotspot settings

  • For more information, visit http://www.android.com/tether#wifi
