USB 網絡共享

使用 USB 數據線令電腦和手機一起共享網絡，電腦就能共享裝置的互聯網連線
  • 網絡共享可在 Windows 7、Windows Vista 和 Linux 上使用

  • 網絡共享時無法連接SD卡到電腦

  • 如需更多資訊，包括如何在其他操作系統上使用 USB 網絡共享，請前往http://www.android.com/tether#usb

重新啟動便攜式 Wi-Fi 熱點

可以將裝置轉換為便攜式 Wi-Fi 熱點，與電腦或其他裝置共用互聯網連接
  • 當裝置作為便攜式 Wi-Fi 熱點使用時，無法透過 Wi-Fi 連接以使用裝置應用程式存取互聯網

  • 可以在 Wi-Fi 熱點設定中配置便攜式 Wi-Fi 熱點

  • 如需更多資訊，請瀏覽 http://www.android.com/tether#wifi
