USB-tethering

Du kan tether din telefon til din computer med et USB-kabel for at dele din telefons internetforbindelse med din computer
  • Tethering fungerer med Windows Vista, Windows 7 og Linux

  • Du kan ikke montere din telefons SD-kort på computeren, når USB er tethered

  • Få flere oplysninger om bl.a. brug af USB-tethering med andre operativsystemer på http://www.android.com/tether#usb

Wi-Fi-tethering

Du kan gøre din telefon til et Wi-Fi-adgangspunkt, så du kan dele din telefons internetforbindelse med en eller flere computere eller andre enheder
  • Når din telefon fungerer som adgangspunkt, kan du ikke bruge programmerne på din telefon til at få adgang til internettet via dens Wi-Fi-forbindelse.

  • Du kan konfigurere et Wi-Fi-adgangspunkt i indstillingerne for Wi-Fi-tethering

  • Få flere oplysninger på http://www.android.com/tether#wifi
