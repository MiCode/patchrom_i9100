USB тіркеу

USB кабелімен байланған кезде компьютеріңіз құрылғыңыздағы интернет байланысын бөлісе алады
  • Windows 7, Windows Vista және Linux жүйелеріндегі байлау жұмыстары

  • Байланған кезде SD картасын компьютеріңізге жалғау мүмкін емес

  • Басқа операциялық жүйелермен USB арқылы байлауды пайдалану жолдарын қамтитын қосымша ақпарат алу үшін http://www.android.com/tether#usb сайтына өтіңіз

Портативті Wi-Fi "жедел" нүктесі

Интернет байланысын компьютерлермен немесе басқа құрылғылармен бөлісу үшін құрылғыңызды Wi-Fi ыстық нүктесіне айналдыруға болады
  • Құрылғы портативті Wi-Fi ыстық нүктесі ретінде қолданылғанда Wi-Fi байланысы арқылы интернетке кіру үшін құрылғы бағдарламаларын пайдалану мүмкін емес

  • Портативті Wi-Fi ыстық нүктесін Wi-Fi ыстық нүкте параметрлерінен конфигурациялауға болады

  • Толық ақпарат алу үшін %s бетіне өтіңіз http://www.android.com/tether#wifi
