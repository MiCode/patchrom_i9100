USB-tilknytning

Du kan knytte telefonen til datamaskinen med en USB-kabel, slik at du kan dele telefonens Internett-tilkobling med maskinen
  • Tilknytning fungerer med Windows Vista, Windows 7 og Linux

  • Du kan ikke montere telefonens SD-kort i datamaskinen når telefonen er tilknyttet via USB

  • Du finner mer informasjon, herunder om hvordan du bruker USB-tilknytning med andre operativsystemer, på http://www.android.com/tether#usb

Tilknytning av trådløst nett

Telefonen kan angis som et tilgangspunkt for trådløst nett for deling av Internett-tilkobling med én eller flere datamaskiner eller andre enheter
  • Når telefonen fungerer som tilgangspunkt, kan du ikke bruke telefonens programmer til å få tilgang til Internett via trådløst nett

  • Gå til innstillinger for tilknytning av trådløst nett for å konfigurere tilgangspunkt for trådløst nett

  • Du finner mer informasjon på http://www.android.com/tether#wifi
