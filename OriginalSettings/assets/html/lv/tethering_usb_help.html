USB piesaiste

Jūsu dators var koplietot ierīces interneta savienojumu, ja vien starp datoru un ierīci izveidosiet savienojumu, izmantojot USB kabeli
  • Piesaistīšana darbojas operētājsistēmā Windows 7, Windows Vista un Linux

  • Piesaistes laikā datoram nevar pievienot SD karti

  • Lai iegūtu plašāku informāciju, kā arī lai uzzinātu, kā lietot USB piesaisti citās operētājsistēmās, apmeklējiet vietni http://www.android.com/tether#usb
