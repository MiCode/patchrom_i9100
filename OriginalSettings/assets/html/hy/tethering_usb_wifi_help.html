USB tethering

Ձեր համակարգիչը կարող է համօգտագործել Ձեր հեռախոսի ինտերնետային միացումը, եթե դրանք տեղակապեք USB կաբելով
  • Տեղակապումը գործում է Windows 7, Windows Vista և Linux համակարգերով

  • Դուք չեք կարող SD քարտ միացնել Ձեր համակարգչին, երբ այն տեղակապված է

  • Լրացուցիչ տեղեկությունների համար, ներառյալ այն մասին, թե ինչպես USB տեղակապում օգտագործել այլ գործութային համակարգերի հետ, այցելեք http://www.android.com/tether#usb

Portable Wi-Fi hotspot

You can turn your device into portable Wi-Fi hotspot, to share its Internet connection with computers or other devices
  • Unable to use device applications to access Internet via Wi-Fi connection while device serves as portable Wi-Fi hotspot

  • You can configure portable Wi-Fi hotspot in Wi-Fi hotspot settings

  • For more information, visit http://www.android.com/tether#wifi
