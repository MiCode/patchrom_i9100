USB tethering

คุณสามารถเชื่อมต่อโทรศัพท์ กับคอมพิวเตอร์กับสาย USB เพื่อแบ่งบันการเชื่อมต่อ อินเตอร์เน็ตของโทรศัพท์ไปยังคอมพิวเตอร์
  • การ Tether ใช้งานได้กับ Windows Vista, Windows 7 และ Linux

  • คุณไม่สามารถติดตั้ง SD การ์ดของคุณบนคอมพิวเตอร์ได้ เมื่อใช้งาน tether

  • สำหรับข้อมูลเพิ่มเติม รวมถึงการใช้ USB Tethering กับ OS อื่นๆ ดูที่
