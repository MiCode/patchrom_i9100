การ​เชื่อมโยงเครือข่ายด้วย USB

คอมพิวเตอร์​ของ​คุณ​สามารถ​แชร์​การ​เชื่อม​ต่อ​อินเตอร์​เน็ต​ของ​อุปกรณ์​ได้ หาก​คุณ​เชื่อมโยง​ไว้​กับกับ USB
  • การ​เชื่อมโยง​เครือข่ายกับ Windows 7, Windows ​Vista ​และ Linux

  • คุณ​ไม่​สามารถ​เชื่อม​ต่อ SD ​การ์ด​ใน​คอมพิวเตอร์​ได้​ขณะ​ที่เชื่อมโยง​อยู่

  • สำหรับ​ข้อมูล​เพิ่ม​เติม​รวม​ถึง​วิธี​ใช้​การ​เชื่อมโยง​ด้วย USB ​กับ​ระบบ​ปฏิบัติ​การ​อื่นๆ ให้​ไป​ที่ http://www.android.com/tether#usb

ฮอตสปอ​ต​ Wi-Fi ​แบบ​พก​พา

คุณ​สามารถ​เปลี่ยน​อุ​ปกรณ์​เป็น​ฮอตสปอ​ต​ Wi-Fi ​แบบ​พก​พา เพื่อ​แชร์​การ​เชื่อม​ต่อ​อินเตอร์​เน็ต​กับ​คอมพิวเตอร์​หรือ​อุปกรณ์​อื่น​ได้
  • ไม่​สามารถ​ใช้​แอ​พพลิ​เค​ชั่นอุปกรณ์​เพื่อ​เข้า​สู่​อินเตอร์​เน็ต​ผ่าน​การ​เชื่อม​ต่อ Wi-Fi ​ใน​ขณะ​ที่​อุปกรณ์​ทำ​หน้า​ที่​เป็น​ฮอตสปอ​ต​ Wi-Fi ​แบบ​พก​พา

  • คุณ​สามารถ​กำหนด​ค่า​ฮอตสปอ​ต​ Wi-Fi ​แบบ​พก​พา​ได้​ใน​การ​ตั้ง​ค่า​ฮอตสปอ​ต​ Wi-Fi

  • สำหรับ​ข้อมูล​เพิ่ม​เติม ให้​ไป​ที่ http://www.android.com/tether#wifi
