USB-internettdeling

Datamaskinen kan dele enhetens Internett-forbindelse hvis du kobler dem sammen med en USB-kabel
  • Internettdeling fungerer med Windows 7, Windows Vista og Linux

  • Du kan ikke koble et SD-kort til datamaskinen ved internettdeling

  • For mer informasjon, inkludert hvordan du bruker USB-internettdeling med andre operativsystemer, går du til http://www.android.com/tether#usb

Portabelt Wi-Fi-hotspot

Du kan gjøre om enheten til et portabelt Wi-Fi-hotspot for å dele Internett-forbindelsen med datamaskiner eller andre enheter
  • Kan ikke bruke enhetsprogrammer for å få tilgang til Internett via Wi-Fi-forbindelse mens enheten fungerer som portabelt Wi-Fi-hotspot

  • Du kan konfigurere portabelt Wi-Fi.hotspot i innstillinger for Wi-Fi-hotspot

  • Gå til http://www.android.com/tether#wifi for mer informasjon
