USB-internetmegosztás

USB-kábel segítségével megoszthatja az eszköz internetkapcsolatát a számítógéppel
  • Az internetmegosztás Windows 7, Windows Vista és Linux operációs rendszerrel működik

  • Internetmegosztás közben nem csatlakoztatható SD-kártya a számítógéphez

  • További információkért, beleértve az USB-internetmegosztás használatát egyéb operációs rendszerekkel, látogasson el a http://www.android.com/tether#usb

Mobil Wi-Fi hotspot

Az eszközt mobil Wi-Fi hotspotként használva, megoszthatja internetkapcsolatát számítógépekkel vagy egyéb eszközökkel
  • Amikor az eszköz mobil Wi-Fi hotspotként működik, az eszköz alkalmazásaival nem lehet Wi-Fi kapcsolaton keresztül hozzáférni az internethez

  • A mobil Wi-Fi hotspotot a Wi-Fi hotspot beállításaiban konfigurálhatja

  • További információt itt találhat: http://www.android.com/tether#wifi
