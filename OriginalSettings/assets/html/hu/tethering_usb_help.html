USB-internetmegosztás

USB-kábel segítségével megoszthatja az eszköz internetkapcsolatát a számítógéppel
  • Az internetmegosztás Windows 7, Windows Vista és Linux operációs rendszerrel működik

  • Internetmegosztás közben nem csatlakoztatható SD-kártya a számítógéphez

  • További információkért, beleértve az USB-internetmegosztás használatát egyéb operációs rendszerekkel, látogasson el a http://www.android.com/tether#usb
