USB bağlanıyor

Bir USB kablosu ile bağladıysanız bilgisayarınız cihazınızın Internet bağlantısını paylaşabilir
  • Bağlanma işlemi Windows 7, Windows Vista ve Linux ile çalışır

  • Bağlı olduğunda bilgisayarınıza SD kartı takamazsınız

  • USB bağlantısının diğer işletim sistemleriyle nasıl kullanılacağı dahil daha fazla bilgi için, http://www.android.com/tether#usb adresine gidin

Taşınabilir Wi-Fi alanı

İnternet bağlantısını bilgisayarlar veya diğer cihazlarla paylaşmak için cihazınızı taşınabilir Wi-Fi alanına dönüştürebilirsiniz
  • Cihaz taşınabilir Wi-Fi alanı olarak çalışırken cihaz uygulamaları Wi-Fi bağlantısı üzerinden internete erişim için kullanılamıyor

  • Wi-Fi alanı ayarlarından taşınabilir Wi-Fi alanlarını yapılandırabilirsiniz

  • Daha fazla bilgi için http://www.android.com/tether#wifi adresini ziyaret edin
