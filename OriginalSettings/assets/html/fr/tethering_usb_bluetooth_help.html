Modem USB

Vous pouvez brancher votre périphérique Android sur votre ordinateur à l'aide d'un câble USB afin de pour partager la connexion Internet de votre périphérique avec votre ordinateur
  • Le modem USB fonctionne avec Windows Vista, Windows 7 et Linux

  • Si votre périphérique est équipé d'une carte SD ou d'un stockage USB, vous ne pouvez pas le brancher sur votre ordinateur lorsqu'il est connecté via USB

  • Pour plus d'informations, notamment sur l'utilisation du modem USB avec d'autres systèmes d'exploitation, voir http://www.android.com/tether#usb

Modem Bluetooth

Vous pouvez brancher votre périphérique Android sur votre ordinateur via une connexion Bluetooth afin de pour partager la connexion Internet de votre périphérique avec votre ordinateur
  • Le modem Bluetooth fonctionne avec n'importe quel ordinateur ou autre périphérique pouvant accéder à Internet via Bluetooth

  • Pour plus d'informations, voir http://www.android.com/tether#bluetooth
