Φορητό σημείο πρόσβασης Wi-Fi

Μπορείτε να μετατρέψετε τη συσκευή σας σε φορητό σημείο πρόσβασης Wi-Fi για την κοινή χρήση της σύνδεσης στο Internet με υπολογιστές ή άλλες συσκευές
  • Δεν είναι δυνατή η χρήση των εφαρμογών της συσκευής για την πρόσβαση στο Internet μέσω Wi-Fi, όταν η συσκευή χρησιμοποιείται ως φορητό σημείο πρόσβασης Wi-Fi

  • Μπορείτε να διαμορφώσετε το φορητό σημείο πρόσβασης Wi-Fi από τις ρυθμίσεις φορητού σημείου πρόσβασης Wi-Fi

  • Για περισσότερες πληροφορίες, επισκεφθείτε τη διεύθυνση http://www.android.com/tether#wifi

Tethering μέσω USB

Είναι δυνατή η κοινή χρήση της σύνδεσης Internet της συσκευής σας με τον υπολογιστή σας, εάν συνδεθούν με καλώδιο USB
  • Η λειτουργία tethering είναι συμβατή με Windows 7, Windows Vista και Linux

  • Μετά την εκτέλεση tethering δεν μπορείτε να συνδέσετε κάρτα SD στον υπολογιστή σας

  • Για περισσότερες πληροφορίες, συμπεριλαμβανομένου του τρόπου χρήσης της λειτουργίας tethering μέσω USB με άλλα λειτουργικά συστήματα, μεταβείτε στη διεύθυνση http://www.android.com/tether#usb

Tethering μέσω Bluetooth

Για να χρησιμοποιήσετε από κοινού τη σύνδεση Internet της συσκευής με τον υπολογιστή, μπορείτε να συνδέσετε τη συσκευή Android στον υπολογιστή σας μέσω μιας σύνδεσης Bluetooth
  • Το tethering μέσω Bluetooth λειτουργεί με οποιονδήποτε υπολογιστή ή συσκευή που μπορεί να συνδεθεί στο Internet μέσω Bluetooth

  • Για περισσότερες πληροφορίες, ανατρέξτε στη διεύθυνση http://www.android.com/tether#bluetooth
