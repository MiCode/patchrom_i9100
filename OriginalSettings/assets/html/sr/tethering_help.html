USB vezivanje

Računar može da deli Internet vezu uređaja ako ih povežeš USB kablom
  • Vezivanje je moguće kod sistema Windows 7, Windows Vista i Linux

  • Na računar se ne može priključiti SD kartica tokom udaljenog vezivanja

  • Za informacije o tome kako se USB vezivanje koristi sa drugim operativnim sistemima idi na http://www.android.com/tether#usb

Prenosiva Wi-Fi pristupna tačka

Uređaj možeš pretvoriti u prenosivu Wi-Fi pristupnu tačku radi deljenja njegove Internet veze sa računarima i drugim uređajima
  • Nije moguće koristiti aplikacije uređaja za pristup Internetu preko Wi-Fi veze dok uređaj služi kao prenosiva Wi-Fi pristupna tačka

  • Možeš da konfigurišeš prenosivu Wi-Fi pristupnu tačku u podešavanjima Wi-Fi pristupne tačke

  • Dodatne informacije potraži na http://www.android.com/tether#wifi
