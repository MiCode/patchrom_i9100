Tethering USB

El seu equip pot compartir la connexió a Internet del dispositiu si els connecta amb un cable USB
  • El Tethering funciona amb Windows 7, Windows Vista i Linux

  • No es pot connectar la targeta SD a l'equip quan està connectat

  • Per obtenir més informació, com ara com s'utilitza el tethering amb altres sistemes operatius, consulti http://www.android.com/tether#usb

Punt calent de Wi-Fi portàtil

Pot convertir el seu dispositiu en un punt calent portàtil Wi-Fi, per compartir la connexió a Internet amb ordinadors o altres dispositius
  • No es poden utilitzar aplicacions del dispositiu per accedir a Internet amb una connexió Wi-Fi mentre el dispositiu funciona de punt calent portàtil Wi-Fi

  • Pot configurar el punt calent portàtil Wi-Fi als ajustaments de punt calent Wi-Fi

  • Per obtenir més informació, visiti http://www.android.com/tether#wifi
