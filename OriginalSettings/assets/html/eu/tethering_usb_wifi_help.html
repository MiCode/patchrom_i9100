USB ainguraketa

Zure ordenagailuak zure telefonoko Interneteko konexioa parteka dezake USB kablearekin ainguratzen badituzu
  • Ainguraketak Windows 7, Windows Vista eta Linux bertsioekin funtzionatzen du

  • Ezin duzu SD txartel bat zure ordenagailura konektatu ainguratu ondoren

  • Informazio gehiago izateko, baita USB ainguraketa beste sistema eragile batzuekin erabiltzeko moduari buruzkoa ere, joan helbide honetara http://www.android.com/tether#usb

Portable Wi-Fi hotspot

You can turn your device into portable Wi-Fi hotspot, to share its Internet connection with computers or other devices
  • Unable to use device applications to access Internet via Wi-Fi connection while device serves as portable Wi-Fi hotspot

  • You can configure portable Wi-Fi hotspot in Wi-Fi hotspot settings

  • For more information, visit http://www.android.com/tether#wifi
