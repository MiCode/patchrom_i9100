Tethering USB

È possibile condividere la connessione Internet di un dispositivo Android con il proprio computer eseguendo il tethering con un cavo USB
  • Il tethering USB funziona con Windows Vista, Windows 7 e Linux

  • Se il dispositivo dispone di una scheda SD o una memoria USB, non è possibile montarlo sul computer se collegato tramite tethering con USB

  • Per ulteriori informazioni, incluse quelle relative all'utilizzo del tethering USB con altri sistemi operativi, visitare il sito Web http://www.android.com/tether#usb

Tethering Bluetooth

È possibile condividere la connessione Internet di un dispositivo Android con il proprio computer eseguendo il tethering con una connessione Bluetooth
  • Il tethering Bluetooth funziona con qualsiasi computer o altro dispositivo in grado di stabilire una connessione Internet con Bluetooth

  • Per ulteriori informazioni, visitare il sito Web http://www.android.com/tether#bluetooth
