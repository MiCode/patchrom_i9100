Vínculo USB

Você pode vincular seu telefone ao seu computador com um cabo USB, compartilhando a conexão com a internet do telefone
  • O vínculo funciona com Windows Vista, Windows 7 e Linux

  • Você não pode usar o cartão SD do telefone no computador com vínculo USB

  • Para mais informações, incluindo o uso de vínculo USB em outros SOs, veja http://www.android.com/tether#usb

Vínculo Wi-Fi

Use seu telefone como um ponto de acesso Wi-Fi, compartilhando a conexão com a internet do telefone com computadores ou outros aparelhos
  • Quando seu telefone atua como ponto de acesso, você não pode usar os aplicativos do telefone para acessar a internet via conexão Wi-Fi

  • Configure um ponto de acesso Wi-Fi nas configurações de vínculo Wi-Fi

  • Para mais informações, visite http://www.android.com/tether#wifi
