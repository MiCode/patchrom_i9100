Furnizare acces la Internet prin USB

Computerul dvs. poate partaja conexiunea la Internet a dispozitivului dvs. dacă le interconectaţi utilizând un cablu USB
  • Partajarea conexiunii la Internet funcţionează cu Windows 7, Windows Vista şi Linux

  • Nu puteţi conecta cartela SD la computer dacă se partajează conexiunea la Internet

  • Pentru mai multe informaţii, inclusiv modul în care se utilizează partajarea conexiunii la Internet prin USB cu alte sisteme de operare, accesaţi http://www.android.com/tether#usb
