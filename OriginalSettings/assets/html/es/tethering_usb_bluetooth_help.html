Módem USB

Para compartir la conexión de su dispositivo con su ordenador, puede conectar como módem su dispositivo Android a su ordenador a través de un cable USB
  • El módem USB funciona con Windows Vista, Windows 7, and Linux

  • Si su dispositivo tiene una tarjeta SD o un almacenaje USB, puede conectarlo a su equipo cuando utilice el anclaje de red mediante USB

  • Para obtener más información, incluida información de uso del dispositivo como módem USB con otros sistemas operativos, consulte http://www.android.com/tether#usb

Anclaje de red de Bluetooth

Para compartir la conexión de su dispositivo con su ordenador, puede conectar como módem su dispositivo Android a su ordenador a través de una conexión Bluetooth
  • La conexión a red Bluetooth funciona con cualquier equipo u otro dispositivo que pueda acceder a Internet mediante Bluetooth

  • Para obtener más información, consulte http://www.android.com/tether#bluetooth
