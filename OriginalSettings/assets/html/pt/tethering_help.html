Ancoragem USB

O seu computador pode partilhar a ligação de Internet do seu dispositivo se os ancorar um ao outro com o cabo USB
  • A ancoragem funciona com o Windows 7, Windows Vista e Linux

  • Não pode ligar o cartão SD ao seu computador quando este estiver ancorado

  • Para mais informações, incluindo sobre como utilizar a ancoragem USB com outros sistemas operativos, vá a http://www.android.com/tether#usb

Hotspot Wi-Fi portátil

Pode transformar o seu dispositivo num hotspot Wi-Fi portátil para partilhar a respectiva ligação à Internet com computadores ou outros dispositivos
  • Impossível utilizar aplicações do dispositivo para aceder à Internet através da ligação Wi-Fi enquanto o dispositivo estiver a servir de hotspot Wi-Fi portátil

  • Pode configurar o hotspot Wi-Fi portátil nas definições de hotspot Wi-Fi

  • Para mais informações, visite http://www.android.com/tether#wifi
