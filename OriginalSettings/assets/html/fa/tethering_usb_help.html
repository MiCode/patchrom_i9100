اتصال USB

اگر رایانه و تلفن را با استفاده از کابل USB به هم وصل کنید، رایانه شما می تواند از اتصال اینترنت تلفن شما استفاده کند.
  • اتصال به اینترنت از طریق تلفن همراه در Windows Vista، Windows 7 و Linux کار می کند

  • شما نمی توانید کارت SD را در هنگام اتصال به اینترنت از طریق تلفن همراه به رایانه خود وصل کنید

  • برای کسب اطلاعات بیشتر، از قبیل چگونگی استفاده از اتصال اینترنتی USB با سیستم عامل های دیگر، به http://www.android.com/tether#usb
