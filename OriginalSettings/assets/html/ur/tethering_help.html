پورٹیبل Wi-Fi ہاٹ اسپاٹ

آپ اپنے آلہ کو پورٹیبل Wi-Fi میں پلٹ سکتے ہیں، کمپیوٹروں یا دیگر آلات کو اس کے انٹرنیٹ کنکشن سے شراکت کے لئے
  • جب آلہ بطور پورٹیبل Wi-Fi ہاٹ اسپاٹ خدمت انجام دیتا ہے تو آلہ کی ایپلیکیشنز کو براستہ Wi-Fi کنکشن انٹرنیٹ پر رسائی کے لئے استعمال کرنے سے قاصر

  • آپ پورٹیبل Wi-Fi ہاٹ اسپاٹ کی تشکیل Wi-Fi ہاٹ اسپاٹ سیٹنگز میں کرسکتے ہیں

  • مزید معلومات کے لئے، تشریف لائیں http://www.android.com/tether#wifi

USB پالہنگ

آپ کا کمپیوٹر آپ کے آلہ کے انٹرنیٹ کنکشن میں شریک ہوسکتا ہے اگر آپ ان کو باہم USB کیبل سے پالہنگ کردیں
  • پالہنگ کرنا Windows 7, Windows Vista، اور Linux کے ساتھ کام کرتا ہے

  • جب پالہنگ کیا گیا ہو تو آپ SD کارڈ کو اپنے کمپیوٹر سے نہیں جوڑ سکتے

  • مزید معلومات کے لئے، بشمول دوسرے آپریٹنگ سسٹموں کے ساتھ USB پالہنگ کرنا کیسے استعمال کیا جائے، ملاحظہ کریں http://www.android.com/tether#usb

بلیوٹوتھ پالہنگ

آپ براستہ بلیوٹوتھ اپنے Android آلہ کو اپنے کمپیوٹر سے پالہنگ کرسکتے ہیں، اپنے کمپیوٹر سے اس کے انٹرنیٹ کنکشن میں شراکت کے لئے
  • بلیوٹوتھ پالہنگ کسی کمپیوٹر یا آلہ کے ساتھ کام کرتا ہے جو انٹرنیٹ پر براستہ بلیوٹوتھ رسائی کرسکتا ہو

  • مزید معلومات کے لئے، تشریف لائیں http://www.android.com/tether#bluetooth
