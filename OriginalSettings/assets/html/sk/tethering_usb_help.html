Zdieľanie pripojenia USB

Počítač môže zdieľať internetové pripojenie zariadenia, ak ich spolu prepojíte na zdieľanie pripojenia pomocou kábla USB
  • Zdieľanie pripojenia funguje so systémami Windows 7, Windows Vista a Linux

  • Počas zdieľania pripojenia nemôžete pripojiť kartu SD k počítaču

  • Ďalšie informácie vrátane toho, ako použiť zdieľanie pripojenia cez USB s inými operačnými systémami, nájdete na stránke http://www.android.com/tether#usb
