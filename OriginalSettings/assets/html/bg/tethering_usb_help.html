USB привързване

Компютърът ви може да споделя връзката с Интернет на устройството, ако ги сдвоите с USB кабел
  • Сдвояването работи с Windows 7, Windows Vista и Linux

  • Не можете да свързвате SD карта към компютъра, когато е сдвоен

  • За повече информация, включително за използването на USB сдвояване с други операционни системи, отидете в http://www.android.com/tether#usb
