שימוש כמודם סלולרי באמצעות USB

תוכל לקשר בין מכשירך למחשב שלך באמצעות כבל USB, בכדי לשתף את רשת האינטרנט הסלולרי של מכשירך עם המחשב
  • השימוש במודם סלולרי באמצעות USB עובד עם Windows 7, Windows Vista ו-Linux

  • אין אפשרות לחבר כרטיס SD למחשב שלך בזמן שימוש כמודם סלולרי באמצעות USB

  • למידע נוסף, כמו איך להשתמש במודם סלולרי USB עם מערכות הפעלה אחרות, עבור אל http://www.android.com/tether#usb
