שימוש כמודם סלולרי באמצעות USB

המחשב שלך יכול להשתמש ברשת האינטרנט הסלולרית של המכשיר שלך, ע"י חיבור מכשירך למחשב באמצעות כבל USB
  • השימוש במודם סלולרי באמצעות USB עובד עם Windows 7, Windows Vista ו-Linux

  • אין אפשרות לחבר כרטיס SD למחשב שלך בזמן שימוש כמודם סלולרי באמצעות USB

  • למידע נוסף, כמו איך להשתמש במודם סלולרי USB עם מערכות הפעלה אחרות, עבור אל http://www.android.com/tether#usb

Mobile AP

מכשירך יכול להפוך לנקודת Wi-Fi ניידת, בכדי לשתף את האינטרנט הסלולארי שלך עם מחשבים או מכשירים אחרים
  • אין אפשרות להשתמש באפליקציות המכשיר באמצעות חיבור Wi-Fi כאשר המכשיר משמש כנקודת Wi-Fi ניידת

  • תוכל לשנות את הגדרות נקודת ה-Wi-Fi הניידת בהגדרות Mobile AP

  • למידע נוסף, בקר באתר http://www.android.com/tether#wifi
